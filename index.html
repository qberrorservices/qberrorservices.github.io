How to use the Network issues tab?

Here's a general guide on how to use the Network Issues tab in the QuickBooks Tool Hub:

  • Open QuickBooks Tool Hub:
    • If you haven't already, download and install the QuickBooks Tool Hub from the official Intuit website.

  • Launch the Tool Hub:
    • Double-click on the QuickBooks Tool Hub icon to open the application.

  • Navigate to the Network Issues tab:
    • Once the Tool Hub is open, look for and click on the "Network Issues" tab. This tab is specifically designed to address problems related to network connectivity.

  • Run Network Diagnostic Tool:
    • Within the Network Issues tab, you may find a tool called the "QuickBooks Database Server Manager" or a similar network diagnostic tool. Click on the relevant option to run the tool.

  • Follow on-screen instructions:
    • The tool will guide you through a series of steps to diagnose and fix network-related issues. Follow the on-screen instructions carefully.

  • Select your QuickBooks version:
    • You may be prompted to select your version of QuickBooks. Make sure to choose the correct version to ensure the tool addresses the specific requirements.

  • Scan the company file:
    • The tool may ask you to select the company file that is experiencing network issues. Choose the appropriate file and let the tool scan for any problems.

  • Review results:
    • After the scan is complete, review the results. The tool will provide information about any network issues it detected and attempt to fix them.

  • Restart QuickBooks and check connectivity:
    • Once the tool has completed its process, restart QuickBooks and check if the network issues have been resolved.

  • Seek additional support if needed:
    • If the network issues persist or if the tool indicates that further assistance is required, consider reaching out to QuickBooks support or consulting the official documentation for more advanced troubleshooting steps.
